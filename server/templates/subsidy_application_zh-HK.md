# 前海港澳青年補貼申請文件模板（繁體中文）

## 申請資格摘要

### 基本條件
1. **身份要求**：45歲以下、有中國國籍嘅香港永久性居民
2. **學歷要求**：至少有副學士或高級文憑學歷
3. **就業/創業要求**：
   - **就業人士**：同前海合作區內嘅公司簽訂一年以上勞動合同，並連續交社保至少6個月
   - **創業者**：在公司中持有至少25%嘅股份，並連續交社保至少6個月

### 補貼類型與金額
- **就業補貼**：根據學歷發放（學士每月¥3,000），最多3年
- **居住補貼**：每月¥1,000，申請人及家屬在深圳無自有住房，最多3年
- **生活補貼**：每月¥2,000，最多3年

**總計：每月最高可獲¥6,000補貼，持續3年**

## 所需文件清單

### 1. 身份證明
- [ ] 香港永久性居民身份證（掃描件）
- [ ] 港澳居民來往內地通行證（回鄉證）（掃描件）

### 2. 學歷證明
- [ ] 最高學歷嘅畢業證書及學位證書（掃描件）
- [ ] 如係海外學歷，需提供教育部留學服務中心出具嘅學歷學位認證書

### 3. 就業/創業證明

**就業人士需提供：**
- [ ] 同前海公司簽訂嘅勞動合同（掃描件）

**創業者需提供：**
- [ ] 公司營業執照（掃描件）
- [ ] 公司章程或最新工商變更備案通知書（證明股權比例≥25%）

### 4. 社保繳納證明
- [ ] 深圳社保局網站或「i深圳」App下載嘅個人參保繳費明細
- [ ] 需顯示在前海連續繳納至少6個月

### 5. 銀行戶口證明
- [ ] 申請人本人在內地開立嘅銀行戶口卡/摺（掃描件）
- [ ] 需顯示戶口號碼

### 6. 個人承諾書
- [ ] 在申請系統下載模板填寫
- [ ] 承諾資料真實、符合申請條件（例如在深圳無自有住房等）

### 7. 個人陳述（可選但建議提供）
- [ ] 突出個人背景、創業/就業動機、對前海發展嘅貢獻計劃

## 線上申請步驟

### 第一步：準備資料
按照上述清單準備所有文件嘅電子掃描件，確保文件清晰可讀。

### 第二步：登錄系統
留意「**前海港澳e站通**」或前海管理局官網發布嘅年度申報通知，在指定時間內登錄申報系統。

### 第三步：填寫資料
進入線上申報系統後，按要求填寫：
- 個人基本信息（姓名、身份證號、聯絡方式）
- 公司信息（如適用）
- 社保繳納記錄
- 住房情況

### 第四步：上傳文件
將所有準備好嘅文件按系統指示上傳，注意：
- 文件格式通常為 PDF 或 JPG
- 單個文件大小不超過 5MB
- 確保文件命名清晰（例如：身份證_張三.pdf）

### 第五步：提交申請
確認所有資料無誤後提交申請，系統會生成申請編號，請妥善保存。

### 第六步：等待審核
- 初審：通常5-10個工作日
- 如資料不齊，會通知你在限期內（通常5個工作日）補交
- 保持電話暢通，及時查收郵件和系統通知

### 第七步：公示與撥付
- 審核通過後，會在前海管理局官網公示5個工作日
- 公示無異議後，補貼款項會直接存入你提供嘅內地銀行戶口
- 首次撥付可能需要1-2個月，後續按月發放

## 重要提醒

### 時間節點
- 申報時間：通常每年3-5月，具體以官方通知為準
- 社保要求：申請前需已連續繳納至少6個月
- 補貼發放：審核通過後按月發放，最長3年

### 常見問題
1. **Q：我可以同時申請多項補貼嗎？**
   A：可以，符合條件可同時申請就業、居住、生活補貼。

2. **Q：創業者如何證明股權比例？**
   A：提供公司章程或最新工商變更備案通知書即可。

3. **Q：補貼會影響我在香港嘅稅務嗎？**
   A：建議咨詢專業稅務顧問，了解兩地稅務影響。

4. **Q：如果中途離職或公司註銷，補貼會停止嗎？**
   A：會，補貼發放以持續符合條件為前提。
